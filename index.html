Craft Your Voxel Worlds: Introducing Pixzor

Pixzor is a free, open-source voxel editor developed in C++ that empowers you to create detailed 3D environments,
landscapes, and objects using an intuitive voxel-based approach.

View the Source Code on GitHub

Future Roadmap

  • 1. Core Voxel Editing Tools and Basic UI
    • Basic voxel placement and deletion
    • Simple color and texture application
    • Sparse voxel octree data structure for larger voxel worlds
    • Fundamental UI framework (window management, basic widgets)
    • File operations UI (New, Open, Save, Save As)
  • 2. Basic Scripting System and Script Editor
    • Integration of Lua scripting language
    • Basic API for voxel manipulation and world interaction
    • Script management system (load, unload, reload)
    • Simple script editor with syntax highlighting
  • 3. Simple Timeline and Animation UI
    • Basic keyframe-based animation editor
    • Simple camera controls and animation
    • Timeline UI with basic playback controls
  • 4. Terrain Sculpting Tools and UI
    • Height-based terrain editing
    • Simple brush system
    • Basic vegetation instancing
    • UI for terrain tool selection and settings
  • 5. Advanced Scripting, Procedural Generation, and UI Enhancements
    • Extended API for complex voxel manipulations
    • Procedural structure generation (buildings, landscapes)
    • Basic physics simulations
    • Enhanced script editor with debugging capabilities
    • UI for managing and applying procedural generation scripts
  • 6. Enhanced Animation, Dynamics, and UI
    • Advanced keyframing and animation blending
    • Particle systems
    • Basic rigid body dynamics
    • Improved timeline UI with advanced animation controls
    • UI for particle system and physics settings
  • 7. Advanced Features and Comprehensive UI (Long-term goals)
    • Terrain erosion simulations
    • Organic shaping with deformation brushes
    • Character rigging with inverse kinematics
    • AI-assisted rendering and motion stylization
    • Comprehensive, customizable UI system
    • User preference and project settings UI
